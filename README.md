
# Quick Start
~~~
 $ mkdir ./data/certs
 cp <pki directory from `fioctl keys ca create`>/factory_ca.pem ./data/certs/
 cp <pki directory from `fioctl keys ca create`>/local_ca.pem ./data/certs/
 cp <pki directory from `fioctl keys ca create`>/local_ca.key ./data/certs/
 cp <pki directory from `fioctl keys ca create`>/tls-crt ./data/certs/
~~~
